# 🛒 E-Commerce

A full-stack E-Commerce web application built using the MERN stack. It includes a customer-facing shopping website, an admin dashboard, and a backend API for managing products, users, and orders.

## 🚀 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

```text
frontend/   # Customer Website
admin/      # Admin Dashboard
backend/    # Server & API
```

## ⚙️ Installation

```bash
git clone https://github.com/lalbabukumar12/E_commerce-.git
cd E_commerce-
```

Install dependencies:

```bash
cd frontend && npm install
cd ../backend && npm install
cd ../admin && npm install
```

Run the project:

```bash
# Backend
cd backend
npm start

# Frontend
cd frontend
npm run dev

# Admin
cd admin
npm run dev
```
